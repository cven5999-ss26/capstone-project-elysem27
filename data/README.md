## Methodological information

1.  Description of methods used for collection/generation of data: load profile
information (kWh, kWp) were gathered primarily by smart meters as part of the
grid's distribution network. That data is aggregated by the local IT team
via AWS.

2.  Methods for processing the data: The data comes in a very raw format 
that needs to be summarized and filtered by category, date/time, and 
consumption to get a better understanding of typical values to help size
other micro-grids.
