Introduction

This project explores existing power consumption data over a set range of dates from April - May 2026. The data was gathered from an operational solar mini-grid in the DRC providing power to a local community. The goal of the project is to summarize the consumption by category to be able to scale the expected consumption for other sites to predict mini-grid sizing in conflict-ridden areas with minimal on-the-ground data collection.

Methods

After receiving the initial data from the IT team that was received from smart meters, the raw data was examined and processed.
